📂 CARICAMENTO DEL DATASET

Scegli file

Apre un file JSON dal tuo disco. Il file deve contenere un array di domande nel formato PegasoQuiz.

PDF → Quiz

Carica un PDF di dispense e genera domande automaticamente tramite intelligenza artificiale.

⚠ Richiede una chiave OpenAI — inseriscila nelle impostazioni avanzate (vedi sezione Easter Egg).

  • Modalità Quick — elabora le prime 3 pagine, genera max 12 domande. Utile per un test rapido.
  • Modalità Complete — elabora tutto il PDF, genera fino a 400 domande (circa 3 per pagina).

💡 Il JSON generato viene salvato accanto al PDF originale e può essere ricaricato in futuro senza nuove chiamate AI.

Carica URL

Scarica e carica un file JSON da un indirizzo web. Supporta URL GitHub anche in formato blob (convertiti automaticamente in raw).

Usa JSON

Incolla direttamente il contenuto JSON nell’area di testo.

Cloud Picker

Accede alla libreria di dataset ospitata su GitHub. Richiede login con username e password (forniti dal docente o dall’amministratore). Dopo il login clicca sul nome del dataset per caricarlo.

☑ SELEZIONE CAPITOLI

Dopo il caricamento appare la griglia dei capitoli. Spunta quelli che vuoi includere nel quiz.

  • Seleziona / Deseleziona — spunta o deseleziona tutti i capitoli in un click.
  • Distribuzione percentuale — quando attiva puoi impostare per ogni capitolo la percentuale di domande da includere.
    Esempio: Lezione 1 = 40%, Lezione 2 = 60% su 10 domande     → 4 dalla Lezione 1, 6 dalla Lezione 2.

Il contatore Bacino X/Y nella barra in basso mostra quante domande hai già estratto (usate) su quante disponibili per la selezione corrente.

🎯 AVVIARE UN QUIZ

Quiz [nome dataset]

Genera un quiz con le domande del dataset. Nel popup scegli quante domande vuoi, oppure seleziona Tutte.
Le domande vengono pescate dal bacino delle non ancora usate: ogni sessione copre domande diverse. Quando il bacino si esaurisce riparte automaticamente dall’inizio.

Modalità esame

Attiva un timer da 30 minuti e introduce le seguenti restrizioni:

  • Il tasto Correggi tutto si sblocca solo dopo 15 minuti.
  • Ogni 2 minuti appare un popup di check presenza da confermare per continuare.

Quiz errori (N)

Genera un quiz usando solo le domande che hai sbagliato nelle sessioni precedenti. Il numero tra parentesi indica quante domande errate hai in archivio per questo dataset. Disponibile solo se hai almeno un errore salvato.

📝 DURANTE IL QUIZ

Ogni card mostra la domanda (sfondo rosso) e le risposte sotto. Clicca su una risposta per selezionarla — puoi cambiare idea fino alla correzione. Il contatore Risposte X/Y e il timer sono visibili nella barra delle azioni.

Correggi tutto

Disponibile solo dopo aver risposto a tutte le domande. Mostra il punteggio e visualizza ogni domanda con:

  • ■ Verde — risposta corretta
  • ■ Rosso — risposta sbagliata (viene mostrata la corretta)

Le domande sbagliate vengono automaticamente aggiunte all’archivio errori per sessioni future.

Apri PDF  (compare accanto alle domande sbagliate)

Apre il PDF delle dispense alla pagina esatta del capitolo corrispondente. Richiede il file .qmap nella stessa cartella del JSON.

Interroga A.I.  (compare accanto alle domande sbagliate)

Chiede a GPT-4o-mini di spiegare perché la risposta era sbagliata e come evitare l’errore. Richiede la chiave OpenAI (vedi Easter Egg).

Indietro

Chiude il quiz senza salvare il risultato e torna alla schermata principale.

📊 STATISTICHE

Il pannello a destra mostra lo storico delle sessioni per il dataset corrente. Ogni riga riporta data, risultato e percentuale.

  • Click su una riga → apre lo Snapshot: dettaglio completo della sessione con tutte le domande, le risposte date e quelle corrette. Da qui puoi riaprire il PDF o chiedere spiegazioni AI anche a posteriori.
  • X (bottone rosso) → elimina la statistica dopo conferma.

Il pannello Statistiche Errori (in alto, bordo rosso) mostra le sessioni della modalità Quiz errori, separatamente dal quiz normale.

🗑 GESTIONE ERRORI E BACINO

Stampa errori

Apre un’anteprima di stampa con tutte le domande del tuo archivio errori, raggruppate per capitolo.

Reset bacino X/Y  (barra in basso)

Azzera il contatore delle domande già usate per la selezione corrente. Il prossimo quiz potrà pescare di nuovo tutte le domande dall’inizio.

Reset bacino errori (N)  (barra in basso)

⚠ Operazione irreversibile. Cancella definitivamente tutte le domande salvate come errate per questo dataset.

📐 PROFILI DISTRIBUZIONE PERCENTUALE

Quando la modalità Distribuzione percentuale è attiva puoi salvare la configurazione corrente come profilo con nome.

  • Salva → salva il profilo con il nome inserito nel campo testo
  • Carica → applica un profilo salvato in precedenza
  • Elimina → rimuove il profilo selezionato
  • Applica → conferma le percentuali e chiude il pannello

I profili sono legati al dataset: un profilo creato con il Dataset A non appare con il Dataset B. Con il login Cloud i profili sono sincronizzati su tutti i dispositivi.

☁ SINCRONIZZAZIONE CLOUD

Dopo il login tramite Cloud Picker, statistiche, errori e profili vengono salvati sul server e sono disponibili da qualsiasi dispositivo con lo stesso account.

Senza login tutto viene salvato localmente in:

  • macOS / Linux: ~/.quiznova/state.json
  • Windows: C:\Users\<utente>\.quiznova\state.json

🔧 PROBLEMI FREQUENTI

Problema

Soluzione

PDF → Quiz è grigio

Manca la chiave OpenAI. Aprire l’Easter Egg e inserire la chiave nel campo AI Key.

Quiz errori è grigio

Non ci sono errori salvati. Completa almeno un quiz con qualche risposta sbagliata.

Cloud Picker non mostra dataset

Verifica di essere loggato e che il Manifest URL nell’Easter Egg sia corretto.

Errore SSL al caricamento URL

Aggiorna l’applicazione o segnala all’amministratore (certificati CA da aggiornare).

Bacino mostra 0/0

Seleziona almeno un capitolo dalla griglia.

Statistiche non visibili dopo login

Le statistiche sono associate al dataset caricato. Ricarica lo stesso dataset delle sessioni precedenti.

📋 FORMATO JSON DEI DATASET

Per creare o modificare un dataset manualmente usa questo formato:

[

  {

    "question":    "Testo della domanda?",

    "chapter":     "Lezione 3",

    "choices": [

      { "text": "Risposta A", "correct": false },

      { "text": "Risposta B", "correct": true  },

      { "text": "Risposta C", "correct": false },

      { "text": "Risposta D", "correct": false }

    ],

    "explanation": "Spiegazione opzionale."

  }

]

In alternativa le risposte possono essere stringhe semplici con il suffisso §§§[OK] sulla risposta corretta. I campi accettano alias: chapter = capitolo / lezione  •  question = domanda / text.
