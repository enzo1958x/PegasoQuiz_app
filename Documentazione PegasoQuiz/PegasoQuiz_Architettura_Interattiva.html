Chiavi di Lookup

qid — identità domanda
id se non "Q###", altrimenti "H" + SHA256(question.lower())[:16]
dataset_key — identità dataset (content-based)
SHA256("|".join(sorted(qid+"#"+chapter for each item)))[:16]
used_key — bacino domande estratte
dataset_hash + ":" + ",".join(sorted(selected_chapters)).lower()
_item_index — lookup O(1)
dict[str → QuizItem] precalcolato da _rebuild_item_index() dopo ogni load
